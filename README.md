# React Isomorphic Starterkit
React.js &amp; Node.js - javascript that run on client-side and server-side

## Installation
```bash
git clone https://github.com/DarekPobozniak/react-isomorphic.git
cd react-isomorphic
npm install
```

#### Development
```bash
npm run watch
```

#### Production
```bash
npm run serve
```
